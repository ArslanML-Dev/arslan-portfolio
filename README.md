## 🚀 Overview

A fully responsive, single-page personal portfolio built from scratch using **pure HTML, CSS, and Vanilla JavaScript** — no frameworks, no dependencies. Designed to make a strong first impression with a dark glassmorphism aesthetic, fluid micro-animations, and a clean developer identity.

> **Live →** [arslan-portfolio-lac.vercel.app](https://arslan-portfolio-lac.vercel.app)

---

## ✨ Features

| Feature | Details |
|---|---|
| **Typewriter Hero** | Animated role cycling — *Data Science Enthusiast, Python Learner, ML Explorer* |
| **Skill Cards** | Animated progress bars with honest self-assessment across 4 categories |
| **3D Tilt Effect** | Perspective-based card tilt on mouse move for depth and interactivity |
| **Magnetic Buttons** | Cursor-following button motion for a premium feel |
| **Scroll-Triggered Reveals** | Intersection Observer powers section fade-ins as you scroll |
| **Cursor Glow** | Real-time ambient glow that tracks cursor position |
| **Floating Particles** | Procedurally generated hero background particles |
| **Ripple Click Effect** | Material-style ripple on all CTA buttons |
| **Contact Form** | Client-side validated form with real-time feedback |
| **Fully Responsive** | Mobile-first layout with hamburger navigation |

## 🎨 Design Highlights

- **Dark glassmorphism** UI with layered blur, transparency, and gradient accents
- **Color palette** — Cyan (`#06b6d4`) × Violet (`#7c3aed`) × Indigo (`#4f46e5`)
- **Typography** — Outfit (headings) & Plus Jakarta Sans (body)
- **Ambient blobs** — CSS-animated gradient orbs for atmospheric depth
- Smooth scroll, active nav highlighting, and sticky transparent navbar
